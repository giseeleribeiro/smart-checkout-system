# Smart Checkout System

A simple prototype developed as part of a university project focused on improving the online shopping checkout experience.

The main goal of this project was to design and build a smarter and more user-friendly checkout flow for an e-commerce platform. The idea came from common problems faced by online shoppers, such as confusing navigation, abandoned carts, and long checkout processes.

This project was created collaboratively for a case study called *OneStop*, an e-commerce business operating across different market segments (B2B, B2C, C2C, and C2B).

---

## Project Overview

The Smart Checkout System was designed to simulate a real online shopping journey, guiding users through:

- Homepage
- Shopping Cart
- Shipping Information
- Payment Page
- Review Order
- Order Confirmation

The focus of the project was not only functionality, but also creating a cleaner and smoother user experience.
